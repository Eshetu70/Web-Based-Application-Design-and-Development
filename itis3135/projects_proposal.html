Project Proposal

Project Overview

This website will be designed for a tutorial giver, specifically Mr. Eshetu, to store various parts of his tutorial center. The intended user of the website will be student who wants to learn back-end and front-end programming languages. The website will have a navigation bar for various programming language courses such as Java, Python, HTML, CSS, and JavaScript and a list of course rules or academic integrity, contact information, about me, worksheets, and possibly a calendar for the daily students.
